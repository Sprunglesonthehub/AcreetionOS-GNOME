# AcreetionOS GNOME - Repository Mirror

This repository serves as a mirror for the AcreetionOS GNOME flavour of the AcreetionOS Linux distribution.

## 🌟 What is AcreetionOS?

AcreetionOS is an **Arch Linux-based distribution designed specifically for newcomers** to the Linux world. While Arch Linux is renowned for its power, flexibility, and rolling-release model, it traditionally has a steeper learning curve during installation and configuration.

AcreetionOS aims to bridge this gap by providing:

*   **Ease of Installation:** A simplified installation process compared to vanilla Arch.
*   **Pre-configured Environment:** Sensible defaults and necessary drivers pre-installed to get you up and running quickly.
*   **Arch Power, Newbie Friendly:** You get the benefits of Arch (latest software, AUR access, rolling release) without the initial hurdles.

## 🖥️ AcreetionOS GNOME

AcreetionOS GNOME is the secondary edition, featuring the **GNOME desktop environment**. GNOME is known for its modern, user-friendly interface, robust application ecosystem, and intuitive workflow, making it an excellent choice for those new to Linux.

## 🚀 Getting Started with AcreetionOS GNOME

1.  **Download the ISO:** Browse this repository and download the latest AcreetionOS GNOME ISO file.
2.  **Create Bootable Media:** Use a tool like Etcher, Rufus (Windows), or `dd` (Linux/macOS) to write the ISO image to a USB drive (8GB or larger recommended).
3.  **Boot from USB:** Restart your computer, enter your BIOS/UEFI settings (usually by pressing F2, F10, F12, or DEL during boot), and set your computer to boot from the USB drive.
4.  **Install:** Follow the on-screen instructions provided by the AcreetionOS installer.

For detailed installation guides and further documentation, please refer to the [original project homepage](https://darrengames.ddns.net:2500/sprungles/acreetonos-gnome).
